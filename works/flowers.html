Ptit cadoau flowers

スクール課題で健康にまつわる架空サイト「Ptit cadoau flowers」を作成しました。
お花と健康をかけ合わせ、統一感のあるサイトにしています。コロナ化でもオンラインでお花を購入でき、おうちでも美しいお花に囲まれながら心身ともに健康的な毎日を過ごせることを伝えられるように工夫しました。

ターゲット
美しいものを好み、美容と健康に自己投資する20代後半～40代前半の独身女性
目的
  • ・お花が美容と健康に効果があることを知っていただき、オンライン購入をしていただくため
  • ・ワークショップに参加していただき、定期的に参加してくださるファンを増やすため
情報設計
ハンバーガーボタン内ではオンラインショップとインスタグラムだけ目立たせ、オンラインショップにすぐリンクして購入しやすくさせたり、インスタグラムからお花のギャラリーを見ていただいて、DMからでも気軽に問い合わせができるような造りにしました。
ニュースとコラムで2つにわけ、活動内容を文章としても写真としても閲覧できるようにしています。
今回重要なオンラインショップとインスタグラムをfotter内でアイコンにせず、しっかりボタンにして目立たせ、クリックを促します。
デザイン
サイトを見た時に感覚的に「美しい」と感じていただくために、まずは視覚的に刺激と衝撃を与えたいと考えました。ローディングではショップ名をアウトライン化し、何が出てくるのかわくわく感を持たせ、その後に画面いっぱいに美しいお花を表示さました。紫にはリラックス効果があるため、お花＝このサイトを見ると落ち着くという感情を狙いました。
設定したターゲットが興味を持つようにヨガスクールやインテリアショップ、レストランとのワークショップをコラムに盛り込んでいます。お花単体として目立たせるのではなく、運動・食事・住まいに関連させ、日々の生活にお花を取り込めるように写真や文章を考えました。
制作期間
1週間
使用ツール
Illustrator / Photoshop / Figma / Visual Studio Code
